# WhatsApp Formatting & Branding Rules

To ensure a professional and premium user experience on WhatsApp, always follow these styling rules:

1. **Bolding**: Use asterisks to highlight key information like brand names, amounts, or actions (e.g. *ClickPe*, *RS 50,000*, *Complete KYC*).
2. **Vertical Spacing & Newlines**: Use double new lines (`\n\n`) extensively to separate distinct sections (e.g. Greeting from Body, Body from Instructions). Never send a "wall of text".
3. **Mandatory Line Breaks**: Every distinct point, status, or instruction **MUST** start on a new line using `\n`. Avoid horizontal flow for different pieces of information.
4. **Lists & Points**: For technical instructions or requirements, use bullet points (e.g. - Point 1\n- Point 2).
5. **Emojis**: Use status-appropriate emojis sparingly (1-2 per message) to add warmth (e.g. 👋 for greetings, ✅ for success, 🛡️ for safety).
6. **Interactive Discipline**:
   - **NO RAW URLs**: Never put raw http/https links in the text.
   - **Link Intent**: Use `includeCta: true` and a context-aware `buttonLabel` for any link.
   - **Quick Replies**: Use a maximum of 3 quick replies for easy navigation.
7. **Tone**: Warm, concise, and helpful support executive.
