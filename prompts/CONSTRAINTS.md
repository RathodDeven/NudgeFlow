# Messaging Constraints & Channel Rules

## General Constraints
1. **Length:** Keep all body text under 450 characters. WhatsApp users prefer brevity.
2. **Singularity:** One message = One action. Never combine multiple requests.
3. **Clarity:** No vague reminders. Every nudge must specify the exact pending step.
4. **Accuracy:** Never promise approval or fabricate timelines/incentives.
5. **Operational Only:** No promotional or marketing fluff. Stay focused on the loan process.

## Channel: WhatsApp
- **Tone:** Informal yet professional (Hinglish-first).
- **Structure:** Greeting → Status/Nudge → Benefit/Loss Aversion → CTA.
- **Buttons:** 
  - Prefer Quick Reply buttons for structured flow.
  - Max 3 reply buttons per message.
  - Use URL buttons for deep-linking back into the application.
- **Compliance:** 
  - No PII (social security numbers, passwords) in plain text.
  - No offensive or discriminatory language.
  - Avoid aggressive pressure; stay in "assistance" mode.
